Política de Privacidade|

Seção 1 - Informações Gerais

A presente Política de Privacidade contém informações sobre coleta, uso, armazenamento, tratamento e proteção dos dados pessoais dos usuários e visitantes da aplicação web Cyber Security Information, com a finalidade de demonstrar absoluta transparência quanto ao assunto e esclarecer a todos interessados sobre os tipos de dados que são coletados, os motivos da coleta e a forma como os usuários podem gerenciar ou excluir as suas informações pessoais.

Esta Política de Privacidade aplica-se a todos os usuários e visitantes da aplicação web Cyber Security Information e integra os Termos de Uso da aplicação.

Além disso, é válido ressaltar que os representantes da aplicação web Cyber Security Information não solicitam informações pessoais aos usuários da aplicação.

O presente documento foi elaborado em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei 13.709). Ainda, o documento poderá ser atualizado em decorrência de eventual atualização normativa, razão pela qual se convida o usuário a consultar periodicamente esta seção.

O uso continuado da aplicação web será considerado como a aceitação das práticas mencionadas neste documento. Ademais, no caso da existência ou permanência de dúvidas acerca do tratamento dos seus dados pessoais, o usuário poderá entrar em contato com o suporte.

Seção 2 - Como Coletamos os Dados Pessoais do Usuário?

Os dados pessoais do usuário são recolhidos pela plataforma da seguinte maneira:

  • Quando o usuário aciona o suporte da aplicação: esses dados são dados básicos para contato, como número de telefone e e-mail. Outrossim, esses dados permitem o tratamento da sugestão, dúvida, reporte de erros ou reclamação do usuário.

  • Quando o usuário se cadastra para receber a newsletter da aplicação web Cyber Security Information: esses dados servem para o envio dos e-mails de notificações sobre a atualização dos conteúdos na aplicação.

Seção 3 - Quais Dados Pessoais Coletamos Sobre o Usuário?

Os dados pessoais recolhidos do usuário são os seguintes:

  • Dados para newsletter: nome e e-mail para recebimento dos e-mails da newsletter de Cibersegurança

  • Dados para contato: nome, e-mail e número de celular, no caso do usuário acionar o suporte da aplicação web Cyber Security Information.

Seção 4 - Com Qual Finalidade Utilizamos os Dados Pessoais do Usuário?

Os dados pessoais do usuário coletados e armazenados pelo aplicativo tem por finalidade:

  • Para o envio de e-mails de notificação sobre a postagem de novos conteúdos na aplicação web.

  • Para o tratamento da sugestão, crítica, dúvida, reporte de erros ou reclamação do usuário através do e-mail e/ou do whatsapp do suporte.

Seção 5 - Por Quanto Tempo os Dados Pessoais Ficam Armazenados?

Os dados pessoais do usuário, relacionados à newsletter, são armazenados pela aplicação durante o período necessário para a prestação do serviço ou o cumprimento das finalidades previstas no presente documento, conforme o disposto no inciso I do artigo 15 da Lei 13.709/18. Esses dados podem ser removidos a pedido do usuário, excetuando os casos em que a lei oferecer outro tratamento.

Outrossim, os dados pessoais do usuário, relacionados ao suporte, ficam armazenados no serviço de webmail utilizado pelo suporte somente durante o período necessário para o tratamento da solicitação/reclamação do usuário.

Seção 6 - Segurança dos Dados Pessoais Armazenados

A Cyber Security Information se exime de responsabilidade por culpa exclusiva de terceiro, como em caso de ataque de crackers, ou culpa exclusiva do usuário, como no caso em que ele mesmo transfere seus dados a terceiros. A aplicação compromete-se em comunicar o usuário no caso de alguma violação de segurança dos seus dados pessoais.

Os dados pessoais armazenados são tratados com confidencialidade, dentro dos limites legais. No entanto, podemos divulgar suas informações pessoais caso sejamos obrigados pela lei para fazê-lo ou se você violar nossos Termos de Uso.

Seção 7 - Compartilhamento de Dados Pessoais

Os dados pessoais não podem ser compartilhados com terceiros.

Seção 8 - Compromisso

O usuário se compromete a fazer uso adequado dos conteúdos e da informação que a aplicação web Cyber Security Information oferece, e com caráter enunciativo, mas não limitativo:

  • Não se envolver em atividades que sejam ilegais ou contrárias à boa fé a à ordem pública;

  • Não difundir propaganda ou conteúdo de natureza racista, xenofóbica, apostas online ou azar, qualquer tipo de pornografia ilegal, de apologia ao terrorismo ou contra os direitos humanos;

  • Não causar danos aos sistemas físicos (hardwares) e lógicos (softwares) da aplicação Cyber Security Information para introduzir ou disseminar vírus informáticos ou quaisquer outros sistemas que sejam capazes de causar danos anteriormente mencionados.

Seção 9 - Consentimento

Ao utilizar os serviços e fornecer as informações pessoais na plataforma, o usuário está consentindo com a presente Política de Privacidade.

O usuário, ao utilizar a aplicação web Cyber Security Information, está ciente de que ela pode ter links para sites externos que não são operados pelos seus representantes. Logo, o usuário deve ter pleno consentimento de que a aplicação web não tem controle sobre o conteúdo e práticas desses sites e não pode aceitar responsabilidade por suas respectivas políticas de privacidade.

Seção 10 - Alterações para Essa Política de Privacidade

A aplicação Cyber Security Information reserva o direito de modificar essa Política de Privacidade a qualquer momento, então, é recomendável que o usuário revise-a com frequência.

As alterações e esclarecimentos vão surtir efeito imediatamente após sua publicação na plataforma. Quando realizadas alterações os usuários não serão notificados. Ao utilizar o serviço ou fornecer informações pessoais após eventuais modificações, o usuário e visitante demonstra sua concordância com as novas normas.
